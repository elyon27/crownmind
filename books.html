Book Paths

Choose a path. Apply it daily.

These paths are designed for discipleship: from hearing to doing, from conviction to consistent practice. Start with one path, commit to 30 days, and walk it steadily.

How to use these paths

Pick one path → read 20–30 minutes daily → apply one action step → repeat. If you want help choosing, message us through the Contact page.
