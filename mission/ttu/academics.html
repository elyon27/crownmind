Phase I Academic Standards

  • Modules are completed by lessons + reflection work + short assessments
  • Certificate requires documented completion and verified submission
  • Major curriculum changes pass through advisory review

Future Alignment

Each certificate path is designed to become a credit-bearing course set in Phase II. Phase I archives outcomes for future accreditation documentation.

Rule: Build every module like it will someday appear in a degree catalog.
